About Southern Swallow

Southern Swallow is a technology consultancy, specialising in: 

  • Embedded System design and evaluation
  • Printed Circuit Board design
  • Open source outreach
  • Android software development
  • Energy efficiency research

Recently completed projects

  • 480 processor multi-core system. Design of architecture, PCB, operating system, programming and debug toolchain, codename: 'Swallow'
  • 100 processor ARM + FPGA system using Xilinx Kintex-7 and Zynq. Design of architecture, host PCB and support verilog code.
  • Custom 1GHz 32-bit processor design, tape-out and bring-up, codename 'PAT'.
  • Custom mixed-signal ASIC design for a power supply application.
  • The BEEBS embedded benchmark suite.
  • High performance processor-to-processor interconnects.
  • Energy-optimising real-time operating system for embedded systems.
  • FPGA host for ASIC bring-up and debug. Real-time feedback and scan-chain.
  • Energy-measurement system for embedded platforms. Open-source hardware and software design.
